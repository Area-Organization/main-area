# Architecture & Data Schema

## Folder Structure

The mobile app follows a modular architecture:

```text
apps/mobile/
├── app/                    # Navigation system (Expo Router)
│   ├── (auth)/             # Route group for authentication (Login/Register)
│   ├── (tabs)/             # Main tab-based navigation (Dashboard, Explore)
│   ├── edit-area/          # Screen to edit an automation
│   ├── _layout.tsx
│   ├── +not-found.tsx
│   ├── index.tsx           # Entry point
│   ├── modal.tsx
│   └── oauth-callback.tsx  # Captures the deep link after third-party OAuth login
├── assets/                 # Images, fonts, and splash screen
│   └── images/             # Service icons (Discord, GitHub, etc.)
├── components/             # React Native components (Cards, Custom Inputs, Loaders)
│   ├── ui/                 # User Interface components
│   └── wizard/
├── constants/              # Colors, API URLs, and configuration keys
│   └── theme.tsx
├── hooks/                  # Custom React hooks
│   ├── use-area-wizard.ts
│   ├── user-color-scheme.ts
│   ├── use-color-scheme.web.ts
│   ├── use-service.ts
│   └── use-theme-color.ts
├── lib/                    # Shared logic (API client)
├── scripts/                # Utility scripts for development/build
└── tests/                  # Unit tests (Jest) and integration tests (Testing Library)
```
